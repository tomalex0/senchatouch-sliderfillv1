Senchatouch-v1-sliderfill

Implementing a background color fill when thumb is dragged or value is set.

    Description

    Plugin adds a background fill effect for slider when slider thumb is moved or value is changed.

    How to use ?

    Plugin used with standalone component

    new Ext.form.Slider({
      renderTo: 'someid',
	    value: 5,
	    label: 'Volume',
	    minValue: 0,
	    plugins: [
	        {
	            ptype: 'sliderfill'
	        }
	    ],
	    maxValue: 10
});

    Default css used for fill, change "background-image" property to get desired style

     .x-slider-fill{
        margin:0.925em;
        position:absolute;
        height:0.8em;
        -webkit-border-radius:0.4em;
        border-radius:0.4em;
        margin-top:0.75em !important;
        background-image:-webkit-gradient(linear, left top ,left bottom, from(#0A3A86), color-stop(.5, #4C8DE7), color-stop(.95, #6BABF5), to(#0A3A86));
        z-index:1 ;
    }

    Want to add custom background color ?

    new Ext.form.Slider({
	    renderTo: 'someid',
	    value: 5,
	    cls : 'hotred',
	    label: 'Volume',
	    minValue: 0,
	    plugins: [
	        {
	            ptype: 'sliderfill'
	        }
	    ],
	    maxValue: 10
});
    .hotred .x-slider-fill{
    background-image: -webkit-gradient(linear, 50% 0%, 50% 100%, color-stop(0%, #FDA397), color-stop(2%, #FB3D23), color-stop(100%, #D41C03));
    background-image: linear-gradient(#FDA397,#FB3D23 2%,#D41C03);
}
